[#setup-rosa]
== Setup ROSA

[.console-input]
[source,bash]
----
rosa create cluster --cluster-name="my-rosa-cluster"
----

The command above will create the ROSA cluster in your default AWS region, type `rosa --help` for more information.

More inforamation and options for ROSA setup can be found in the https://docs.openshift.com/rosa/welcome/index.html[official documentation].
